Overview

VGC Calc Formatter is an independent Google Docs add-on that helps users quickly format, structure, and review Pokemon VGC calculation notes inside the active document.

The add-on provides a sidebar interface to insert section headers, classify calculation blocks, and apply consistent visual formatting.

Main Features

  • Insert standardized section headers for offensive and defensive calculation blocks.
  • Format selected text in place inside the active Google Doc.
  • Apply consistent styles for calculation readability.
  • Store per-user formatting preferences using Apps Script PropertiesService.

How It Works

  1. User opens the add-on from the Google Docs add-on button or Extensions menu.
  2. User opens the sidebar and selects a formatting action.
  3. The add-on processes only the current active document and applies formatting.
  4. User preferences are saved for future sessions.

OAuth Scope Usage

Requested scopes and why they are required:

Scope Reason
documents.currentonly Read and modify only the currently active Google Document during user-triggered formatting actions.
script.storage Store user-specific formatting preferences such as colors and thresholds in Apps Script properties.
script.locale Use Google Workspace locale settings to support localized behavior.
script.container.ui Render and control the add-on UI (menu and sidebar) inside Google Docs.
